Story ／ 開発の背景

— 現役行政書士による開発

事務作業の負担軽減によって
より多くの案件を受け入れ、お客様をサポートできるようにしたい。

在留資格業務に携わって6年。クライアント数の増加とともに、ご家族の呼び寄せ・複数案件の並行管理・追加資料対応が重なり、事務作業が業務の大半を占めるようになりました。

Excel、事件簿ソフト、請求書サービス、OneNote ── 複数ツールを駆使しても「確認漏れ」「入力の二度手間」は消えなかった。

「この作業、自動化できないかな？」と、日常の困りごとや解決したいことをひとつずつ機能として反映させていくうちに、現場から生まれたオールインワンツールが完成しました。

DX化で自己申請が増える一方、現場で働く外国人には行政書士の親身なサポートが欠かせない。Clavis Note は、その時間を守るために作りました。

01 / 02 ポイポイ式

書類をまとめてポイッ。
あとは、おまかせ。

在留カード・パスポート・契約書を、ドラッグ＆ドロップするだけで自動読み取り。現場で在留カードの写真を撮れば、そのままクライアント台帳が完成。外国語の退職証明書や履歴書も、AI-OCR でデータ化して、クライアント台帳の9割超を自動登録。入力作業を限りなくゼロに。

※ ポイポイ式は AI を利用した自動読み取り機能です。読み取り内容に誤りがある場合がありますので、最終的な確認は必ず利用者ご自身でお願いします。

All-in-One

主な機能

クライアント台帳

在留カード・パスポート情報、職歴・給与履歴、案件履歴を一元管理。家族構成や勤務先との繋がりも、ひと目でわかる。

  • 家族・勤務先の関連付け
  • 在留期限アラート
  • 変遷履歴の自動記録

案件管理

入管業務がメインだけど、たまにそれ以外の業務も引き受ける ── そんな現場のリアルに応えて、一般業務も管理可能。入管業務には、追加資料対応履歴や審査官・勤務先とのやり取りも記録できます。

  • 事件簿の自動生成
  • 追加資料の対応履歴
  • 関係者ごとのやり取り記録

報酬・入金管理

見積・請求・領収書発行から、まとめ請求、立替金管理まで。現金でいただいた報酬をその場で入金記録、領収書・PDF発行と、現場で完了できる業務フローを実現。

  • 見積→請求→入金 一気通貫
  • 立替金・実費の追跡
  • 確定申告連携データ出力

FAQ

よくある質問

行政書士登録者以外は使えますか?
Clavis Note は行政書士登録者専用のサービスです。一般企業・個人ユーザー・登録支援機関単体の方への提供は行っておりません。これは、在留カード・パスポート等の機微な個人情報を扱う性質上、行政書士法に基づく法定の守秘義務を負う行政書士であることを、日行連（日本行政書士会連合会）のHPで確認させていただくためです。
データはどこに保存されますか?

入力したクライアント情報・案件・報酬データは、お使いのパソコン内に保存され、バックアップデータが Google Drive に同期されます（ご要望により OneDrive 連携の追加も検討します）。当事務所のサーバーには業務データを保管しない設計です。

なお、ご契約者のお名前・メールアドレス・ライセンス情報は、決済・契約管理のため当事務所側で安全に保管しております。詳しくは プライバシーポリシー をご確認ください。

個人開発のようですが、セキュリティは大丈夫ですか?

むしろ、個人開発だからこそ、Clavis Note は、貴事務所の顧客情報を当事務所で預からない設計にしました。

よくあるクラウド型のサービスは、個人情報を「サービス会社の倉庫」のような場所に預けて使う仕組みです。便利な反面、倉庫に預ける仕組み（セキュリティ対策）には高度な専門性が要求されます。守秘義務を負う行政書士業務で使っていただくためには、リスクを最小限にしたいと考えました。

そのため Clavis Note は、その「倉庫」そのものを作らない仕組みになっています。

顧客の個人情報（クライアント台帳・案件・報酬データ・在留カードやパスポートの画像など）は、お使いのパソコン内に保存され、バックアップデータが Google Drive に同期されます（ご要望により OneDrive 連携の追加も検討します）。

当事務所では、業務データを預かることも、仕組み上、閲覧することもできません。例えるなら、紙の台帳を貴事務所のキャビネットに保管しているのと近い感覚です。

ご不明点などがありましたら、info@clavis-note.com までお気軽にお問い合わせください。

ポイポイ式（書類の自動読み取り）は安全ですか?

ポイポイ式は、在留カード・パスポート・雇用契約書などの画像から、必要な情報を抽出してフォームに自動入力する機能です。読み取り処理のため、画像データは外部の AI エンジン（API）を経由します。

  • AI の学習には使われません：API 経由で送信したデータが、AI モデルの学習・改善に利用されることはありません（API 提供事業者の公式仕様）。
  • 不正利用検知のため、一定期間は API 提供事業者側で保管されます：多くの API サービスで採用されている一般的な仕様で、目的外の閲覧・利用は行われません。
  • 手入力でもご利用いただけます：ポイポイ式は「入力をラクにする工夫」のひとつで、利用は任意です。気になる書類はご自身の判断で手入力を選んでいただけます。
使い方はどこで確認したらいいですか?
基本的な操作については、ご利用開始時にお送りする操作ガイドをご参照ください。ご不明な点やお困りごとがありましたら、info@clavis-note.com までお気軽にお問い合わせください。
決済からライセンスキー発行まで時間がかかりますか?
通常は決済完了から数秒～数分でライセンスキーを発行し、welcome メールでお送りしております。ごく稀にシステムの混雑等により10分程度かかる場合があります。30分以上経過しても届かない場合は、info@clavis-note.com までご連絡ください。
welcome メールが届きません
決済完了後、通常は数秒～数分で welcome メールをお送りしております。届かない場合は、まず迷惑メールフォルダをご確認ください。それでも見当たらない場合は、メールアドレスの入力ミスの可能性があります。info@clavis-note.com までお問い合わせいただければ、決済者情報を確認のうえ再送いたします。
「ライセンスキーが無効です」と表示された時は?
ご入力内容に誤りがないかご確認ください。コピー&ペースト時に余分なスペースが混入することがあります。決済直後の場合は、ライセンスキー発行まで数分かかる場合があります。解決しない場合は info@clavis-note.com までお問い合わせください。
解約後、入力したデータはどうなりますか?
解約後は、新規入力・編集は停止されますが、Clavis Note を開いて、過去データの閲覧と事件簿のエクスポートは可能です。
継続を中止したい時はどうすればいいですか?
次回更新日（決済日）の前日までに、所定の解約手続きをお願いいたします。月途中での解約による日割り精算・返金は行っておりません。解約手続きは info@clavis-note.com までご連絡いただくか、Stripe から送付される決済明細メール内のリンクから直接管理いただけます。解約前に、バックアップ機能で業務データを保存してください。
